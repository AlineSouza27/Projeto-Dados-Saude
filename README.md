# Projeto-Dados-Saude

## Contexto
Este projeto tem como objetivo extrair dados epidemiológicos de Dengue disponibilizados pelo governo em formato JSON. Após a extração, os dados serão tratados e armazenados em um banco de dados. Com os dados salvos, realizaremos análises utilizando ferramentas como Power BI (PBIX) e Jupyter Notebook para análises estatísticas e construção de modelos.

## Etapas do Projeto
1. **Extração de Dados**: Obter os dados epidemiológicos em formato JSON.
2. **Tratamento de Dados**: Limpar e transformar os dados para adequação ao banco de dados.
3. **Armazenamento**: Salvar os dados tratados em um banco de dados.
4. **Análise**: Utilizar Power BI e Jupyter Notebook para análises estatísticas e criação de modelos.

## Ferramentas Utilizadas
- Python
- Power BI
- Jupyter Notebook
- Banco de Dados (a definir)
